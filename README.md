# 阿光音樂製作作品集網站

## 使用方式
1. 直接雙擊 `index.html` 即可預覽。
2. 上線時可整個資料夾部署到 GitHub Pages、Netlify、Vercel 或一般虛擬主機。

## 建議優先替換
- 網站名稱與 Logo：`index.html`
- Instagram、Threads、LINE 連結：搜尋「請替換」
- 作品名稱、封面與音檔
- 個人照片：目前使用「光」字形視覺佔位
- 價格與服務說明

## 音檔串接
目前播放器為視覺示意。正式版可新增 `<audio>` 元素，或串接 SoundCloud、YouTube、Spotify。

## 未來 Beat 商店
目前已保留：
- 類型篩選
- BPM / Key 資訊
- 授權按鈕位置
- 商品列表排版

後續可串接 Stripe、綠界、藍新或其他金流，再加入下載權限與授權條款。
